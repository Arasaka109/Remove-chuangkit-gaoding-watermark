# 🎨 创客贴去水印截图辅助脚本

这是一个用于创客贴和稿定网页端编辑器的 Tampermonkey (油猴) 脚本。通过底层拦截、canvas重绘和 DOM 清理，去除编辑区的限制提示与水印，方便用户使用系统自带工具手动截图。

## ✨ 功能特点

- 🛡️ **底层水印拦截**：重写 `URL.createObjectURL` 方法，通过文件大小精准匹配并拦截底层 PNG 水印，将其替换为 1x1 像素的全透明图片，不误伤正常 SVG 和图片素材。
- 🧹 **DOM 干扰清理**：通过 XPath 文本匹配，定时自动移除编辑区阻挡视线的“移除水印，畅享高清模板”浮动按钮。
- ⚡ **极简轻量**：纯原生 JavaScript 实现，不依赖任何第三方截图或渲染库，不拖慢网页运行性能。

## 🚀 安装与使用

1. 🧩 浏览器安装 [Tampermonkey](https://www.tampermonkey.net/) (油猴) 扩展程序。
2. ➕ 在 Tampermonkey 面板中点击“添加新脚本”。
3. 📋 将本项目中的js代码完整复制并粘贴进去，点击保存。
4. 🌐 打开创客贴网页编辑器，脚本将在后台自动运行。
5. 📸 确认画布和中心区域已纯净无干扰后，使用系统截图快捷键（如 `Win+Shift+S` 或 `Cmd+Shift+4`）保存画面。
